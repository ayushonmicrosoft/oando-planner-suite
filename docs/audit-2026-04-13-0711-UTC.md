#### No Content-Security-Policy Header
- **Rule:** `security/csp`
- **Finding:** No `Content-Security-Policy` HTTP header was found. CSP prevents cross-site scripting (XSS), clickjacking, and data injection attacks by specifying which sources of content are allowed.
- **Solution:** Add a CSP header in your Next.js middleware or API layer. Start with a report-only policy to avoid breaking functionality, then tighten it. At minimum, restrict `script-src`, `style-src`, and `default-src`. #### No X-Frame-Options / Clickjacking Protection
- **Rule:** `security/x-frame-options`
